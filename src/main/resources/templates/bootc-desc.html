About the Java Bootcamp

The Java bootcamp is an eight-week intensive introduction to Java programming, as well as the OOP paradigm, front end development, Spring boot framework, and database management. The end goal is knowing how to build a website front to back, in order to build an expanding portfolio on GitHub.

The camp also covers professional development. Students are encouraged weekly to set goals, achieve them, and do them within a social network that expands beyond the college campus. By getting students connected, the potential for future job placement is greatly increased, because a good percentage of jobs are passed along by word of mouth, and those working in companies are often the greatest key to entry.

Students also fill out paperwork that helps them organize their overall strategy and implement it. While some students have never been to a job interview before, or navigated a career before, others receive an expanded perspective on how they might need to interact in an interview.
