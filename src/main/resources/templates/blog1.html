GitHub Individually

2 Feb 2020

GitHub is an extension to git.

Git is version control software which allows one to save their work at any stage of development.

The more stages saved, the more options one has of revisiting a project's potentially superior development; in contrast, without this software one might be forced to spend wasted time undoing bad features.

GitHub takes the locally saved project and displays it on a public site.

This site allows a local host to crash without affecting the quality of software.

Likewise, one can proceed to clone their project on any connected device, freeing up the individual to work from home, at the office, and on vacation.

GitHub Example:

  • git init
  • git add .
  • git commit -m "First Commit"
  • yourGitHubAccount.git
  • git push -u origin master
