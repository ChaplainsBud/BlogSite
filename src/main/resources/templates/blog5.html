HTML

6 Feb 2020

HTML stands for hyper text mark-up language.

HTML "marks-up" the content of a web page.

Elements vary in the amount of hidden functionality they possess.

For example, a 'p' tag simply designates a paragraph, but a video tag enables the display of film. Geolocation is another feature that easily uses a lot of additional code.

Emmet is the best system I am aware of for quickly typing out HTML with a formula.

Pug is the best preprocessor for HTML that I know of, which allows for a template sheet to be written, partials extended, with a minimum amount of work left for unique content.
