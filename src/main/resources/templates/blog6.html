CSS/Bootstrap

10 Feb 2020

CSS stands for cascading style sheets.

While the WWW was originally designed for exchanging scientific documents, it eventually emerged to be useful for commerce, gaming, recreation, and socializing.

CSS adds layout and style features to HTML content, much like a graphic artist's work in the former magazine industry.

It also easily goes back to the history of publishing, making great use of typography.

By increasing the pleasure a user may experience in a site, it invites them to spend more time with material they might otherwise ignore.

For those who find this language to be daunting, Twitter has created a library of CSS and SASS code, as well as jQuery, that makes designing the UI of a site quicker, though more constrained.

CSS Example:

  • *{ padding:0; margin:0;}
  • body { color: green; }
  • footer { display: flex; }
  • Variables have been added to CSS.
  • SASS may still be necessary for loops and functions.
  • Bootstrap has entire components laid out, so you can set up a navbar within minutes.
